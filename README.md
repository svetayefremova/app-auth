# React Native Authorization

Full Stack React Native Authorization using GraphQL and Apollo

## Used the following stack:

- Server - Express
- Client - React Native
- Middleware - Apollo(GraphQL)
- Database - MongoDB

## Additional

- Passport.js
- React Navigation
- Expo

## Usage

- start server

```shell
$ npm run server
```
- go to localhost:3000/graphql

- use client with EXPO app
